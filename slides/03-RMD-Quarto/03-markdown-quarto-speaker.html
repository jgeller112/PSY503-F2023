Equations

An overview

  • Anything inside dollar signs $ is treated as ‚Äúinline‚Äù maths
  • $$ is a standalone equation
  • Whitespace matters: $x$ is an equation, $x $ is not
  • Equations follow ‚ÄúLaTeX‚Äù rules

Equations are special

  • This \(x^2\) is inline

  • This equation is standalone \[\bar{x} = \frac{1}{n} \sum_{i=1}^{n} x_i\]

R markdown advantages

  • Explicitly link R code to output

  • Wide variety of formats

  • Increases transparency

  • Increases efficiency
